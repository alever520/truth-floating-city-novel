# 写作指南与最佳实践

## 一、本书定位

**一句话说清楚：**
> 地球物理学博士穿越到费伦大陆，用科学方法论拆解魔法体系，从学院学徒一路做到位面工业革命的推动者。

核心卖点：**认知差逆袭 + 魔法科学化 + DND 世界观深度融合**

---

## 二、章节结构准则

### 开头铺垫

开头部分一般作为上一章故事的延续，节奏会相对换慢一些

### 发展推进

本章的核心内容，必须推进剧情或深化人物。

**推进方式：**
```
- **新信息揭示**：让读者/主角获得重要信息
- **关系变化**：人物关系发生转变
- **问题升级**：现状恶化，新危机出现
- **角色成长**：主角获得新技能、新认知
```

**避免：**
```
- 纯粹的场景描写（风景、房间布局等）
- 与剧情无关的人物互动
- 重复已知的对话
```

### 结尾钩子

留下悬念，让读者想看下一章。

**典型方式**

#### 紧急危机

角色面临迫在眉睫的危险，下一章必须立即应对。

**示例：**
> 地板开始震动，灰尘从天花板簌簌落下。她抬头一看，裂缝正在迅速扩大。

**关键要素：**
- 时间紧迫
- 威胁明确
- 后果严重

#### 未完成的动作

一个动作被中断，留下"接下来会发生什么"的疑问。

**示例：**
> 他举起枪，手指扣在扳机上——"别动！"身后传来一个声音。

**关键要素：**
- 动作进行中被打断
- 不确定结果
- 新变量出现

#### 身份反转

某人被揭示为不是我们以为的那样。

**示例：**
> "我终于找到你了，弟弟。" 那个说着完美普通话的男人摘下面具，露出了一张她父亲的脸。

**关键要素：**
- 身份误解
- 关系重定义
- 动机重新解读

#### 神秘物品/线索

发现一个重要但意义不明的东西。

**示例：**
> 保险箱里只有一张照片，拍摄于昨天。照片里是熟睡中的她，从窗外角度拍摄。

**关键要素：**
- 物品意义不明
- 暗示威胁
- 激发好奇

#### 言外之意

一句话表面正常，但暗示了更深层的东西。

**示例：**
> "恭喜你通过面试，" 面试官笑着握住她的手，"和你的姐姐一样优秀。" 可她是独生女。

**关键要素：**
- 表面正常
- 隐藏信息
- 需要解读

### 悬念设置

**悬念设置禁忌:**

1. **虚假悬念** - 制造紧张但结果是误会
   > 错误：他听到了脚步声...原来是猫
   > 正确：他听到了脚步声...但追他的人已经死了

2. **机械降神** - 突然出现从未提及的解决方案
   > 错误：她突然想起自己会武术
   > 正确：她想起父亲教过的防身术（第五章提过）

3. **过度留白** - 留下太多未回答问题
   > 原则：每章至少回答一个旧悬念，再提出新悬念

4. **低风险钩子** - 结尾事件不够重要
   > 错误：他不知道晚饭吃什么
   > 正确：他的晚餐被人下了毒

## 三、章节节奏控制原则

### 基本要求
```
- 每章 2000-4000 字左右
- 每章至少完成一件事：推进剧情、加深关系、公开新信息、制造冲突、兑现期待、埋伏笔
- 如果删掉这一章主线不受影响，这章大概率在水
```

### 情绪节奏
```
- 紧张 → 缓和 → 更大紧张 → 回报 → 新问题，循环呼吸
- 小高潮频繁（主角赢一局、解锁新发现、人物关系推进）
- 大高潮蓄力（大战、真相揭露、阶段突破）
```

### 信息密度
```
- **高密度**：动作场面、大量对话、快速事件
- **低密度**：内心独白、环境描写、情感沉淀
```
**原则**：高低密度交替，避免持续高密度（读者疲劳）或持续低密度（读者无聊）

### 章节长度与内容密度对照

| 章节字数 | 核心事件数量 | 场景数量 |
|---------|-------------|---------|
| 800-1500 | 1 个主要事件 | 1-2 个场景 |
| 1500-3000 | 1-2 个主要事件 | 2-3 个场景 |
| 3000-5000 | 2-3 个主要事件 | 3-5 个场景 |

**原则**：每章至少包含一个不可删除的核心事件。如果一个事件可以移除而不影响理解，则应删除。

## 四、章节内容创作原则

### 1. "展示而非讲述"原则

#### ⚠️ 核心原则

这是**最核心也最容易违反**的原则。

#### 执行标准

**严格程度**: 100%严格执行
- ✅ 默认要求:所有情节都要"展示"
- ⚠️ 例外情况:如果后续遇到,再具体补充到本文档
- 📝 原则:遇到不确定的,先按严格标准执行

#### 常见错误示例表

| 讲述(禁止) | 展示(正确) |
|----------|----------|
| 他很愤怒 | 他握紧拳头,指节发白 |
| 她很紧张 | 她反复调整衣角,咬着下唇 |
| 格雷文眼神一凝 | 格雷文瞳孔微缩,手指停在羽毛笔上 |
| 托马斯的表情很复杂——震惊、不解、还有一丝恐惧 | 托马斯张了张嘴,又闭上,手指攥紧了袍子下摆,往后退了半步 |
| 林墨感觉一切都很不真实 | 林墨盯着自己的手,翻来覆去看了三遍,像在确认它们真的属于自己 |

#### 正确做法分类

**情绪 → 用生理反应+动作展示**:
```
- 愤怒: 拳头/呼吸/肌肉紧绷
- 恐惧: 后退/冷汗/瞳孔放大
- 紧张: 攥紧东西/反复小动作
- 震惊: 僵住/张嘴/瞳孔变化
```

**性格 → 用对话+选择展示**:
```
❌ 不要说"他很勇敢"
✅ 而是写"他第一个冲进燃烧的房子"
```

**关系 → 用互动+细节展示**:
```
❌ 不要说"他们是朋友"
✅ 而是写"托马斯守了三天,眼圈都黑了"
```

**内心独白 → 用第一人称或第三人称展示**:
```
方式A:第三方视角(展示)
❌ 林墨很困惑
✅ 林墨皱眉,这不对劲...到底哪里不对?

方式B:第一人称视角(直接写出思考)
✅ 这不对劲。到底哪里不对?
```
**选择标准**: 哪种让文章**更流畅**就用哪种

**环境描写 → 必须避免讲述,让读者有场景感**:
```
❌ 房间很小,很乱
✅ 房间只有十几平米。衣服扔在床上,书本堆满桌子,地上还有几个空餐盘
```
🎯 **目标**: 让读者通过具体细节**感受到场景**,而不是被告知"很小很乱"

#### 6种致命错误（全书禁止,任何章节不得违反）

| 错误类型 | 示例 | 为什么致命 | 例外情况 |
|---------|------|-----------|---------|
| **天气描写** | "那天天气晴朗,万里无云..." | 与故事无关,浪费读者时间 | ✅ 天气影响剧情时可写 |
| **日常流程** | "李明醒来,刷牙洗脸,吃早餐..." | 无聊,没有冲突,无推进 | ✅ 日常中发生异常时可写 |
| **回顾上章** | "上一章我们说到..." | 读者已知,重复信息 | ✅ 通过对话/回忆自然带出 |
| **缓慢铺垫** | "先介绍一下这个城市的背景..." | 信息倾倒,没有行动 | ✅ 通过行动/对话展示 |
| **平淡对话** | "你好,你好吗?我很好。" | 无意义,不推进剧情 | ❌ 无例外 |
| **过度解释** | "这是因为,所以,然后..." | 讲述而非展示 | ✅ 快速过渡时可简述 |

### 2.场景与转场规则

#### 🔴 核心规则

- **同一章节内禁止使用分割线（`---`）做段落转场**
- 分割线仅用于不同人物故事线之间的切换
- 章节内场景过渡靠行为动作、时间跳跃短句、视角自然流转实现
- **禁止简单删除分割线，必须用自然过渡替代**
- **场景自然过渡 = 让读者不感觉到场景在切换**

#### ⚠️ 核心原则

这是**最核心也最容易违反**的原则。

#### 执行标准

**严格程度**: 100%严格执行

#### 五种过渡方式（2026-03-28 新增）

##### 1. **时间推进过渡**

**示例：**
```
托马斯沉默了几秒，最后只是点了点头："那就好。"

过了一会儿，托马斯坚持要林墨吃点东西。
```

**技巧：** 增加"过了一会儿""不久后""几分钟后"，告诉读者时间过去了

##### 2. **动作带场景切换**

**示例：**
```
林墨没再问。但知道托马斯在说谎。

吃完早饭，两人沉默着走回宿舍。路上的学徒渐渐多了起来。

回到宿舍后，林墨终于有时间研究原主的笔记了。
```

**技巧：**
- "吃完早饭" → 食堂场景结束
- "走回宿舍" → 动作带出地点变化
- "回到宿舍后" → 明确新场景开始

##### 3. **状态总结+推进**

**示例：**
```
假设有了，数据分析也完成了。现在需要实际验证。林墨站起来，走到宿舍中央的空地。
```

**技巧：**
- "假设有了，数据分析也完成了" → 总结前一场景
- "现在需要实际验证" → 推进到下一步
- "站起来，走到..." → 动作带出位置变化

##### 4. **环境变化暗示时间流逝**

**示例：**
```
写完最后一个字，林墨放下羽毛笔，抬头看向窗外。光线已经很亮了，太阳升得更高了。
```

**技巧：** 环境变化（光线、温度、声音）= 时间过去了

##### 5. **对话引出+立刻执行**

**示例：**
```
"我陪你去。"

两人推开门，走上石板路。
```

**技巧：**
- "我陪你去" → 决定
- "两人推开门" → 立刻执行，动作带出场景切换

#### 场景过渡检查清单（2026-03-28 新增）

写完一章后，检查每个场景切换点：

- [ ] 删除所有单独成行的`---`分割线
- [ ] 每个场景切换处，是否有过渡句？
- [ ] 场景切换后，读者能否立刻知道"现在在哪里/在做什么"？
- [ ] 两个场景之间，是否有明显的"断裂感"？（有则需加过渡）

### 3. 细节描写原则（沉浸感优先）

#### 🎯 核心理念

**长篇小说不需要节奏过快、信息密度过高**。重点是读者的沉浸感和体验感。

**反模式**：
- ❌ 信息堆砌："房间不大，十几平米。两张单人床，一张简陋书桌。"
- ❌ 抽象形容："头痛欲裂！林墨睁开眼。"
- ❌ 快速推进：3段完成"醒来+记忆涌入+整理身份"

**正确做法**：
- ✅ 用3000字完成原本2000字的内容
- ✅ 每个场景都有视觉/听觉/嗅觉/触觉细节
- ✅ 慢下来，让读者"在场"

---

#### 四大原则

##### 原则1: 具象 > 抽象

**不要用抽象形容，要用具体细节**

| 类型 | ❌ 抽象（禁止） | ✅ 具象（正确） |
|------|---------------|----------------|
| **疼痛** | "头痛欲裂！" | "不是那种钝痛，而是颅底像被什么东西压着，每隔几秒收紧一次，像是有人在脑子里拧螺丝。耳畔传来短促而尖锐的嗡鸣声，就像某种高频信号在脑子里来回扫。" |
| **技术细节** | "屏幕上跳动的异常数值" | "磁场读数异常跳动，从稳定的12.5特斯拉突然飙升到15.2，然后是17.8，数字还在疯狂攀升。警报灯一排排亮起，刺眼的红光在昏暗的实验室里闪烁。" |
| **环境** | "房间很破旧" | "石灰岩砌成的天花板，粗糙的表面上布满裂纹，从中间延伸到角落，像蜘蛛网一样密集。靠窗那一侧还长了浅绿色的霉斑，显然是长期潮湿的结果。" |
| **疲惫** | "托马斯很疲惫" | "托马斯眼睛下面有深深的黑眼圈，脸上写满了疲惫，袖口有些脏，像是好几天没换衣服了。" |

**为什么具象更好**：
- 给读者具体的画面
- 让读者"看见"而不是"被告知"
- 增强代入感和可信度

---

##### 原则2: 演 > 说

**不要直接描述情绪，要通过行为/生理反应"演"出来**

| 情绪 | ❌ 说出来 | ✅ 演出来 |
|------|---------|----------|
| **震惊** | "托马斯震惊地看着林墨" | "托马斯震惊地看着林墨的手心，嘴巴微微张开，眼睛瞪得很大。几秒后，他才找回自己的声音。" |
| **紧张** | "林墨很紧张" | "林墨手心微微出汗。他看着自己的右手。" |
| **怀疑** | "托马斯怀疑林墨的身份" | "托马斯盯着那行字，手指攥紧了袍子下摆。'你……'他开口，声音有些颤抖，'你以前从来不记笔记。'" |
| **冷静思考** | "林墨冷静地分析" | "林墨闭眼，深呼吸，强迫自己冷静。他按导师教过的方法处理：先确认身体感知，再分析信息源。" |

**为什么演更好**：
- 读者自己推断情绪，更有参与感
- 避免"作者在讲述"的旁白感
- 行为更真实，比抽象情绪更可信

---

##### 原则3: 慢下来

**不要急着推进剧情，用细节填充场景**

**❌ 错误示例**（节奏过快）：
```
头痛欲裂！

林墨睁开眼，石灰岩天花板，裂纹从中间延伸到角落。

脑中闪过失去意识前的画面：CERN实验室，粒子对撞机的警报声...

门吱呀一声开了。托马斯端着托盘走进来...
```
**问题**：3段就完成"醒来→回忆→托马斯登场"，读者还没进入状态

**✅ 正确示例**（节奏适中）：
```
头痛欲裂！

不是那种钝痛，而是颅底像被什么东西压着，每隔几秒收紧一次...（200字描写疼痛）

脑中闪过失去意识前的画面——

CERN的地下对撞实验室。冰冷的金属墙壁，控制台上密密麻麻的显示屏...（300字描写事故）

林墨慢慢睁开眼。

石灰岩砌成的天花板，粗糙的表面上布满裂纹...（200字描写房间）

林墨撑起身体，坐起来。动作很慢，每一个关节都在抗议...（100字描写起身）

房间不大，大概十几平方米...（300字继续描写环境）

...（后续继续慢节奏）
```
**效果**：用1200字完成原本400字的内容，沉浸感更强

**为什么慢更好**：
- 长篇小说不是短篇，不需要惜字如金
- 给读者时间"进入"场景
- 细节越多，世界越真实

---

##### 原则4: 有层次

**描写要分层，不要平铺直叙**

**环境描写分层模板**：
1. **第一层**：远景/整体（天花板、墙壁）
2. **第二层**：中景/局部（窗户、家具）
3. **第三层**：近景/细节（霉斑、灰尘）
4. **第四层**：非视觉（气味、声音、温度）

**❌ 错误示例**（平铺直叙）：
```
房间不大，十几平米。两张单人床，一张简陋书桌，墙角堆着几本翻开的书和羊皮纸。窗外传来鸟鸣。空气里有草药的味道。
```
**问题**：信息并列，缺乏层次感，像清单

**✅ 正确示例**（分层描写）：
```
石灰岩砌成的天花板，粗糙的表面上布满裂纹，从中间延伸到角落，像蜘蛛网一样密集。（第一层：天花板）

靠窗那一侧还长了浅绿色的霉斑，显然是长期潮湿的结果。（第二层：窗户附近细节）

窗户是那种中世纪城堡里常见的窄缝式，只有成年人手臂那么宽，外面透进来的光线斜着打在地板上，能清楚看见空气里飘浮的灰尘。（第三层：光线和灰尘）

房间不大，大概十几平方米。墙角胡乱塞着几摞书、皮革卷轴、装着不明液体的小瓶子，还有一些他叫不出名字的植物——有些叶子是紫色的，有些茎干上长着细小的倒刺。（第四层：墙角物品）

空气里弥漫着草药和返潮木头混合的气味，不太好闻，但吸进去后似乎能让人稍微振奋精神，头痛缓解了一些。（第五层：嗅觉+效果）
```
**效果**：像镜头推进，读者跟着视角"扫描"房间

**为什么分层更好**：
- 符合人的真实观察习惯（从大到小、从远到近）
- 引导读者视线，控制节奏
- 避免信息堆砌

---

#### 情绪传递分层模板

1. **第一层**：外在行为（表情、动作）
2. **第二层**：生理反应（心跳、出汗、颤抖）
3. **第三层**：内心独白（思考、判断）

**示例**：
```
托马斯盯着那行字，手指攥紧了袍子下摆。（第一层：行为）

"你……"他开口，声音有些颤抖，"你以前从来不记笔记。"（第二层：生理反应-声音颤抖）

停顿了很久，他才开口，声音很低："你真的还是林墨吗？"（第三层：终于说出内心疑问）
```

---

#### 对话节奏分层

**不要快速问答推进信息，要增加停顿、反应、情绪铺垫**

**❌ 错误示例**（快速问答）：
```
"我昏迷多久了？"
"三天整。"
"月考过了吗？"
"还没！明天上午八点半。"
"考核内容是什么？"
"三环塑能系法术..."
```
**问题**：像背台词，没有情绪波动

**✅ 正确示例**（有层次）：
```
"我昏迷多久了？"林墨问。

"三天整。"托马斯的声音里带着如释重负，"医疗院的人来看过，拿着那个淡蓝色的水晶球在你身上扫了好几遍，说你身体没问题，灵魂波动也稳定了……还好你今天醒了。"

林墨压下心中的疑问。灵魂波动？水晶球检测？先确认最重要的事："月考过了吗？"

"还没！"托马斯眼睛一亮，然后又迅速暗淡下来，"明天上午八点半，你还赶得上。但是……"

他停顿了一下，欲言又止。

"但是什么？"

托马斯移开视线，看向窗外："你昏迷了三天，现在就考试，会不会太勉强了？要不要我去帮你申请延期？"

林墨放下杯子。明天？那我得先搞清楚考什么。

"考核规则是什么？"

托马斯再次愣了一下，这次愣得更久。他盯着林墨看了几秒，像是在确认眼前这个人是不是真的是林墨。

"你……真的没事吗？"托马斯小心翼翼地问，"我是说，你感觉……"

"我没事。"林墨打断他，"我只是想搞清楚考核规则，这样才能有针对性地准备。时间不多了。"

托马斯沉默了几秒，最后还是老实回答："委员会会指定一道三环塑能系法术..."
```
**效果**：
- 有停顿、反应、情绪波动
- 托马斯的关心、犹豫、试探层层展现
- 林墨的冷静、急迫、目标感清晰
- 对话不再是信息交换，而是人物互动

---

#### 比喻选择原则

**不要用烂大街的比喻，要用独特的、精确的**

| 类型 | ❌ 常见比喻 | ✅ 独特比喻 |
|------|-----------|----------|
| **记忆涌入** | "记忆继续涌来，像洪水决堤" | "就像有人拿着漏斗往他脑子里强行倒水，猝不及防，来不及消化" |
| **魔力施法成功** | "林墨感到很轻松" | "就像找到了开锁的正确手法，之前怎么拧都打不开的锁，现在轻轻一转就开了" |
| **意识挣扎** | "意识在黑暗中挣扎" | "林墨的意识在黑暗里挣扎，像溺水的人拼命想浮出水面，却怎么也够不到那道光" |

**为什么独特比喻更好**：
- "洪水决堤"是穿越文烂大街的比喻
- "漏斗倒水"更具象，读者能想象那种被动、不适感
- 好的比喻=精确传达感受+让读者耳目一新

---

#### 施法场景渲染原则

**不要把施法写成"一个动作"，要写成"一个过程"**

**❌ 错误示例**：
```
林墨闭上眼，按原主记忆尝试零环光亮术。

他试着"顺着节奏推"。

手心光芒亮起，比托马斯平时施展的亮将近一倍。
```
**问题**：施法过程被跳过，缺乏渲染

**✅ 正确示例**：
```
托马斯犹豫了一下，最后还是点头："好……你小心点，如果感觉不对立刻停下。我去拿治疗草药。"

他快步走到墙角，从一个小木箱里翻出几包草药放在桌上，然后紧张地站在一旁。

林墨闭上眼，按原主记忆尝试零环光亮术。

意识下沉，感知向内，寻找魔力的流动。

传统的感知方式是"推水"——把魔力当成水流，用意志推动它流向指尖。原主的记忆里，所有教材、所有导师都是这么教的。"与魔法共鸣"，"感受它的流动"，"像推水一样引导它"。

但林墨感觉到的不是水流。

更像是波。

有节奏、有周期、有高峰和低谷。周期大约是1.2秒。

这不是水流，是振动。是能量的周期性变化。

林墨在脑中建立模型：如果魔力是波，那么施法不是推动水流，而是在对的时机推一把——就像推秋千，在波峰时轻轻加力，利用共振放大振幅。

他试着"顺着节奏推"。

手心光芒亮起，比托马斯平时施展的亮将近一倍。

林墨心跳加速——成了！而且他感觉自己几乎没用什么力。

就像找到了开锁的正确手法，之前怎么拧都打不开的锁，现在轻轻一转就开了。

托马斯震惊地看着林墨的手心，嘴巴微微张开，眼睛瞪得很大。几秒后，他才找回自己的声音，试探性地问："你……感觉跟以前一样吗？"
```
**效果**：
- 托马斯的紧张准备（拿草药）增加仪式感
- 林墨的理性分析过程（波vs水流）展现科学家思维
- 成功后的生理反应（心跳加速）增强真实感
- 托马斯的震惊用行为描写（嘴巴张开、眼睛瞪大、找回声音）
- 读者跟着林墨一起经历"发现-尝试-成功"的过程

---

#### 检查清单

写完一章后，检查每个重要场景：

- [ ] 疼痛/情绪描写是否具象？（有没有用"像被压着""每隔几秒"这种细节）
- [ ] 技术细节是否专业？（有没有具体数值、术语）
- [ ] 环境描写是否分层？（天花板→墙→窗→物品→气味，5层递进）
- [ ] 人物情绪是否"演"出来？（有没有行为+生理反应+内心独白三层）
- [ ] 对话是否有停顿和反应？（不要快速问答）
- [ ] 比喻是否独特？（避免"洪水决堤""如同雷鸣"等烂大街的比喻）
- [ ] 重要场景是否渲染充分？（不要一笔带过）
- [ ] 节奏是否适中？（长篇小说可以慢，不要急着推进）

**目标字数参考**：
- 简单场景（日常对话）：800-1000字
- 重要场景（首次施法、关键冲突）：1500-2000字
- 核心场景（月考、重大发现）：2500-3000字

**Remember**: 长篇小说的优势是有足够空间打磨细节，不要浪费这个优势。

## 五、人物塑造规范

**通用原则**: 

- 不需要逐个配角详细定义
- **符合人物性格设定即可**
- 有`characters.md`人物卡时参考

### 人物深度塑造技巧

#### 1. 矛盾性

真实的人是复杂的，充满矛盾。

**示例：**
- 暴力的黑帮成员但爱护流浪猫
- 无神论的牧师
- 害怕黑暗的侦探
- 重视友情但总是背叛朋友

#### 2. 侧面揭示

不要直接陈述性格，通过行为展示。

| 错误（直接陈述） | 正确（侧面展示） |
|----------------|----------------|
| 他很愤怒 | 他捏碎手中的纸杯 |
| 她很紧张 | 她反复调整眼镜位置 |
| 他很傲慢 | 他从不直视下属的眼睛 |
| 她很善良 | 她偷偷喂流浪狗三年 |

#### 3. 声音独特性

每个人说话方式不同，对话中能分辨角色。

**区分要素：**
- 用词选择（正式/俚语/方言）
- 句子长度
- 是否打断别人
- 是否喜欢隐喻
- 情绪表达方式

#### 4. 动机合理化

每个角色行为必须有合理动机，即使动机扭曲。

**反派动机合理化示例：**
- "我想毁灭世界" → 乏味
- "我失去了一切，世界对我没有意义" → 可理解但扭曲
- "人类是地球的病毒，我必须清除" → 有哲学支撑

### 人物出场设计

#### 首次出场原则

**有效的出场方式：**
- **行动中** - 展示能力或性格
- **冲突中** - 立即建立关系/对立
- **误解中** - 建立悬念

**避免：**
- 镜子自照描写外貌
- 姓名+年龄+职业的简历式介绍
- 无意义的日常活动

#### 出场示例对比

| 无效出场 | 有效出场 |
|---------|---------|
| 李明，28岁，是一名侦探。他走进办公室。| 李明跨过警戒线，警官试图拦住他。"市刑警队，李明。"他亮出证件，径直走向尸体。|
| 美丽的女孩坐在窗边，她叫小红。| 她已经三天没睡了，咖啡杯里的液体在颤抖。当门铃响起时，她几乎把杯子摔在地上。|

### 人物语言特征 

#### 主角林墨

**性格定位**: 
- ✅ **情绪和理性平衡**,根据剧情强度调整
- ❌ 不要因为"物理学家"身份而脸谱化
- 🎯 物理学家身份主要体现在**知识和技能**,不限制言行举止
- 🎯 人物言行以**鲜活**为目标

**✅ 应该有的**:
- 理性思维(物理学家背景)
- 用比喻解释(但不是专业术语堆砌)
- 适度的调侃和自嘲
- 对魔法的"科学家视角"
- **根据剧情的情绪变化**:
  - 危机时刻:紧张、恐惧、愤怒
  - 发现新原理:兴奋、好奇
  - 日常时刻:调侃、自嘲、放松

**❌ 不应该有的**:
- 粗口(除非极端情况)
- 网络流行语("该说不说""绝绝子")
- 元叙事表述("狗血的穿越剧情")
- 过度冷静(像在念实验报告)

**🎯 正确示例**:
- ✅ "这种比中彩票还难的事情都能被我碰到吗?"
- ❌ "这种小概率事件都能被我碰到吗?"
- ✅ "那我是运气好还是不好呢?"
- ❌ "该说不说,我运气果然是他妈太好了!"

**示例(不同情境的情绪表达)**:
- 紧张时: "该死,时间不够了!"(情绪化)
- 分析时: "如果魔力流动遵循能量守恒...那这个现象就说得通了。"(理性)
- 日常时: "穿越这种事也能被我碰上?运气还真是好到没边了。"(自嘲)

**使用比喻说明对魔法新的理解**: 每次遇到**新的原理层面**的说明时使用
- ❌ 不是每次遇到新魔法都用
- ✅ 发现新的底层原理/机制时用
- 🎯 双重作用: 推进剧情 + 便于读者理解

**示例**:
- 第一次理解魔力流动: 用"水流"比喻 ✅
- 第二次施展同类魔法: 不需要重复比喻 ❌
- 发现新的底层原理: 再用新比喻 ✅
- 避免过度使用导致重复

#### 其他配角

### 人物一致性检查

角色行为必须符合已建立的性格。

**检查问题：**
- 这件事符合他的核心价值观吗？
- 以他的背景，会有这样的反应吗？
- 他的缺陷会导致他犯什么错？

**例外处理：**
- 如果角色"不符合性格"行事，必须有原因
- 解释应该在相同/下一章提供
- 可以是成长的标志（角色克服缺陷）

## 六、对话写作规范

好对话是揭示人物、推动情节、制造冲突的有力工具。

### 对话核心原则

#### ✅ 好对话的标准

1. **每句对话有目的**: 推动情节/揭示人物/制造冲突
2. **简洁自然**: 删除冗余词语
3. **有潜台词**: 不是所有话都直说
4. **能区分角色**: 不同人说话方式不同

#### ❌ 常见错误

- **无意义对话**: "你好""吃了吗"
- **信息倾倒**: 一次对话说太多背景
- **重复问题**: 可以合并的分开问
- **过度解释**: 读者能理解的不需要说出来

#### 无意义对话的界定

**判断标准**: 对话删掉之后是不是完全没有影响

**完全无意义(必须删除)**:
```
❌ "你好。" "你好。" "天气不错。" "是啊。"
```

**有意义(可以保留)**:
- 体现关系/气氛/互动的对话
- 体现环境/情境的对话
- 推动情节或展示性格的对话

**示例**:
```
✅ "早啊。"托马斯打了个哈欠。
✅ "早。睡得怎么样?"
✅ "别提了,一晚上都在担心你。"
```
- 体现了托马斯的关心和担忧
- 删掉会损失人物关系的展示

### 🎯 对话优化技巧详解

#### **技巧1:合并重复问题**

**核心**: 整合信息,但要注意上下文通顺和连贯

❌ **啰嗦版**:
```
"我昏迷多久了?" "三天。"
"今天几号?" "25号。"
"月考什么时候?" "明天上午。"
"几点开始?" "八点。"
```

✅ **优化版**:
```
"今天几号了?月考过了吗?"
"25号,你昏迷了两天,月考明天上午八点。"
```

**关键点**:
- 不是机械地把所有信息塞进一句话
- 而是**针对关键信息和潜在含义回答**
- 像人正常对话,不一板一眼

#### **技巧2:增加潜台词**

**核心**: 通过回避、转移话题、动作暗示,而不是"说一半留一半"

❌ **太直白(不够潜台词)**:
```
"你……你真的还是林墨吗?"
"什么意思?"
"可能是我多想了。"
```
- 问题:"多想了"已经明说了怀疑,不算暗示

✅ **真正的潜台词**:
```
"你……"托马斯欲言又止,"你真的还是林墨吗?"
林墨心里一紧。"什么意思?"
"没什么。"托马斯移开视线,"我去食堂了。"
```
- "我去食堂了"回避问题,但怀疑已经种下

✅ **更隐晦的版本**:
```
托马斯盯着林墨看了很久。
"怎么了?"林墨问。
"没什么。"托马斯收回目光,"你继续吧。"
```
- 只是盯着看+避而不答,更有潜台词

**关键点**:
- ❌ 不是"说一半"就叫潜台词
- ✅ 而是通过**回避、转移、动作**暗示
- "可能我多想了"(太明显) vs "我去食堂了"(真正的回避)

#### **技巧3:删除无意义客套**

**核心**: 跳过客套,直入主题,用动作和简短对话展示人物

❌ **冗余版**:
```
"格雷文教授,您好。" "你好,林墨。请坐。"
"谢谢。" "要喝水吗?" "不用了,谢谢。"
"那么,你找我有什么事?"
"我想问一下关于魔力流动的问题……"
```

✅ **优化版**:
```
林墨敲门走进办公室。格雷文头也不抬:"说吧,什么事?"
"关于魔力流动,我有些疑问……"
```

**关键点**:
- 格雷文"头也不抬"展示性格(严肃/高冷)
- 跳过客套,直入主题
- 对话推动情节,不浪费篇幅

#### **技巧4:用对话展示性格差异**

**核心**: 不同人物说话方式不同,但避免动作过多造成刻意感

❌ **动作太多,太刻意**:
```
托马斯绞着手指:"要不要……我是说,你可以装病推迟?"
林墨摇头:"推迟只会更可疑。"
"那……"托马斯咬了咬嘴唇,"万一被发现……"
"那就想办法不被发现。"林墨语气平静,但手指在敲桌子。
```
- 问题:绞手指+咬嘴唇+敲桌子,太密集,有刻意感

✅ **精简版,更自然**:
```
"要不要……装病推迟?"托马斯声音很小。
"推迟只会更可疑。"
"那万一被发现……"
"那就想办法不被发现。"林墨手指在桌上轻敲。
```

**关键点**:
- ❌ 不是"越多动作越生动"
- ✅ 一个场景1-2个关键动作足够
- 托马斯的纠结:通过"声音很小"和省略号体现
- 林墨的紧张:只用"手指敲桌"一个动作

#### **技巧5:避免信息倾倒**

**核心**: 信息分散到剧情中,每次只给1-2个信息点

❌ **信息倾倒版**:
```
"学院有很多规则。首先,禁止私斗,违者会被逐出学院。其次,禁止窃取他人研究成果,这是重罪。第三,禁止在非实验区域施展危险法术。第四,每个月都有月考,不及格三次会被开除。第五,实验室需要申请,还要交押金。第六……"
```

✅ **优化版(分散到剧情)**:
```
[第一次]
"小心点,学院禁止私斗。"托马斯压低声音,
"去年有个学徒打伤了同学,直接被逐出学院了。"

[后续章节,实验室场景]
"这里能随便施法吗?"林墨问。
"别。"托马斯赶紧阻止,"非实验区施展危险法术是违规的。"

[再后续,月考前]
"月考很重要吗?"
"当然!不及格三次就会被开除。"
```

**关键点**:
- 规则随剧情自然展开
- 每次只给1-2个信息点
- 结合具体情境,读者更容易记住

#### **技巧6:用对话推进冲突**

**核心**: 对话制造张力,推动剧情

❌ **平淡版**:
```
"你最近进步很快。" "谢谢教授。" "继续努力。"
```

✅ **有冲突版**:
```
"两周前,你连魔法飞弹都释放不出。"格雷文盯着林墨,
"现在你能在三秒内完成。"
"我……找到了窍门。"
"窍门?"格雷文眼神锐利,"还是有人帮了你?"
林墨心跳加速,但表面保持镇定:"我自己摸索的。"
格雷文久久不语,羽毛笔在手指间转动。"是吗。"
```

**关键点**:
- 质疑+压力+暗示不信任
- 林墨的紧张通过"心跳加速"展示
- 格雷文的"是吗"含义丰富
- 推动"学院怀疑"的主线剧情

#### **技巧7:用对话展示关系变化**

**核心**: 保持行文连贯,避免突兀

❌ **太突兀,缺少过渡**:
```
"林墨……"托马斯站在门口。
"怎么了?" "没事,就是……来看看你。"
沉默。
"我先走了。"托马斯转身离开。
```
- 问题:"我先走了"太突然,不自然

✅ **修正版(增加过渡)**:
```
"林墨……"托马斯站在门口,"你还好吧?"
"挺好的。"
托马斯点点头,没再多说,转身离开了。
```
- 点头+没再多说+离开,一气呵成
- 简洁但连贯

✅ **更详细的版本**:
```
"林墨……"托马斯站在门口,没有像往常那样直接进来。
"怎么了?"
"没事,就是……来看看你。"托马斯移开视线,"你还好吧?"
"挺好的。"
沉默。托马斯看了看房间,又看了看林墨,似乎想说什么,
但最终只是点了点头。
"那我不打扰了。"他转身离开。
```
- 增加犹豫和过渡
- "那我不打扰了"比"我先走了"更自然

**关键点**:
- ❌ 对话不能"说完就走",像机器人
- ✅ 需要**自然的过渡**:犹豫、停顿、最后决定
- 即使是疏远,也要"疏远得自然"
- **行文连贯是重中之重**

### 潜台词技巧

**1. 话题转移**
```
"你爱我吗？"
"你看了天气预报吗？明天有雨。"
```

**2. 反问而非回答**
```
"你杀了他吗？"
"你觉得像我这样的人会做那种事？"
```

**3. 谈论其他事物**
```
"你想我吗？"
"我妈昨天打电话来了。"
```

**4. 沉默和动作**
```
"你愿意原谅我吗？"
她继续看杂志，翻了一页。
```

## 七、世界观呈现方式

### 核心原则

**✅ 用细节自然植入**,让读者"感受"而不是"被告知"  
**❌ 避免信息倾倒**,不要直接解释世界观设定

### 方式1:通过道具展示

#### 示例1:魔法道具(恒温法阵)

❌ **错误(直接解释)**:
```
这个世界有恒温法阵,是一种魔法装置,可以保持食物的温度不变。
托马斯用恒温法阵给林墨热粥。
```

✅ **正确(自然植入)**:
```
托马斯递过来那碗一直用恒温法阵热着的粥。
```
- 读者自然理解:这个世界有保温的魔法
- 不需要解释"什么是恒温法阵"

#### 示例2:照明系统(魔晶灯)

❌ **错误(作者视角解释)**:
```
这个世界没有电灯,而是用魔晶灯照明。魔晶灯是一种魔法装置,
由魔力晶石和金属支架组成,可以发出持续的光芒。
```

✅ **正确(通过描写)**:
```
头顶的魔晶灯发出淡蓝色的光,晶石在金属支架中缓缓旋转。
```
- 通过视觉描写,读者看到了照明方式
- 不需要解释工作原理

#### 示例3:货币系统

❌ **错误(角色对角色解释常识)**:
```
"在我们这个世界,货币分为金币、银币和铜币。
1金币等于100银币,1银币等于100铜币。"
```

✅ **正确(通过情节展示)**:
```
林墨掏出钱袋,里面只剩下三枚金币和一把银币。
食堂的饭菜要五个银币,实验材料要十个金币……
按这个速度,撑不过这个月。
```
- 通过"花钱"这个具体行为展示货币
- 读者自然理解金币比银币值钱

### 方式2:通过人物描写展示

#### 示例4:种族特征

❌ **错误(直接陈述)**:
```
这个世界的人类大多是金发碧眼,只有遥远东土的人才有黑发黑眼。
林墨的外貌很罕见。
```

✅ **正确(通过反应展示)**:
```
铜镜里的脸很年轻,黑色的头发,黑色的眼睛。
原主记忆里提到,大陆上大部分人类都是金发碧眼或棕发褐眼,
只有遥远的东土才有这样的特征。
```
- 通过林墨"看镜子+回忆"自然引出
- 暗示他的外貌会引起注意

#### 示例5:社会阶层

❌ **错误(大段背景说明)**:
```
学院里分为贵族学徒和平民学徒。贵族学徒家境优渥,
能买得起昂贵的实验材料,还有家族导师指导。
平民学徒只能靠助学金维持,处境艰难。
```

✅ **正确(通过对比展示)**:
```
走廊尽头,几个贵族学徒正在讨论新到的龙血草。
"我爸给我寄了五株,够我做一整个月实验了。"
林墨低头走过,手里攥着借条——为了买一株龙血草,
他欠了托马斯两个金币。
```
- 通过"对比"展示阶层差异
- 通过"借钱"展示林墨的处境

### 方式3:通过对话展示

#### 示例6:宗教/信仰

❌ **错误(角色解释常识)**:
```
"在我们这个世界,人们信仰梅林,他是最伟大的法师。
每当遇到危险,我们就会祈祷'梅林在上'。"
```

✅ **正确(通过口头禅)**:
```
"梅林在上,还好你今天醒过来了!"托马斯松了口气。
```
- 托马斯的自然反应,不是在"解释"
- 读者自然理解这是这个世界的信仰

#### 示例7:魔法体系

❌ **错误(教科书式讲解)**:
```
"魔法分为九个环阶,学徒最多掌握三环法术。
四环以上需要正式法师的资格。法术释放需要三个要素:
手势、咒语和魔力流动。"
```

✅ **正确(通过具体行为)**:
```
"你能释放几环法术?"
"勉强能放出二环的魔法飞弹。"林墨比划了一下手势,
"但总是不稳定。"
"正常,大部分学徒都卡在二环好几年。"托马斯叹气,
"要升到三环,得在月考上证明自己。"
```
- 通过"具体情境"自然展开
- 信息分散在对话中,不是一次倾倒

### 方式4:通过行动/情节展示

#### 示例8:规则/制度

❌ **错误(规则清单)**:
```
学院有严格的规定:
1. 禁止私斗,违者逐出学院
2. 禁止窃取研究成果
3. 实验事故要赔偿
4. 月考不及格三次会被开除
```

✅ **正确(通过后果展示)**:
```
"小心点,去年有个学徒在走廊里打伤了人。"
托马斯压低声音,"第二天就被赶出学院了。"
林墨记住了:学院禁止私斗。
```

```
"实验室爆炸,要赔多少?"
"一千两百金币。"
林墨倒吸一口凉气。这是他十年的助学金。
```
- 通过"具体案例"展示规则
- 通过"赔偿金额"展示严重性

#### 示例9:魔法原理

❌ **错误(理论讲座)**:
```
魔力本质上是一种能量流动。传统魔法理论认为,
魔力必须按照固定路径流动,这叫"魔力回路"。
每个法术都有对应的标准回路。
```

✅ **正确(通过实践展示)**:
```
林墨伸出手,尝试凝聚魔力。
魔力在掌心聚集,像一团温热的气流。他按照记忆中的路径,
引导魔力沿着手臂流动,在指尖汇聚成形——
一颗拳头大小的光球浮现,摇摇晃晃。
```
- 通过"施法动作"展示魔力流动
- 读者"看到"魔力如何工作,不是被"告知"

### 方式5:通过环境/氛围展示

#### 示例10:学院建筑风格

❌ **错误(旅游指南式描述)**:
```
星辰法师学院是北境最古老的魔法学院,建于三百年前。
学院采用哥特式建筑风格,高耸的尖塔象征着对真理的追求。
```

✅ **正确(通过感官描写)**:
```
林墨走在学院的长廊里,脚步声在高高的拱顶下回荡。
两侧墙壁上挂着历代院长的画像,他们的目光仿佛在审视每一个路过的人。
窗外,高塔的尖顶刺破灰色的天空。
```
- 通过"走路+观察"自然描写
- 读者感受到学院的庄严和历史感

#### 示例11:社会风气

❌ **错误(总结陈述)**:
```
在这个世界,法师拥有崇高的地位,人们都敬畏他们。
平民见到法师都要行礼。
```

✅ **正确(通过细节展示)**:
```
林墨走出学院大门,守卫立刻挺直身体,恭敬地点头致意。
街上的商贩看到他胸前的学徒徽章,连忙让出道路。
一个孩子好奇地盯着他,被母亲一把拉过去:
"不要乱看,那是学院的学徒!"
```
- 通过"他人反应"展示法师地位
- 不需要说"法师很受尊敬"

### ❌ 错误方式总结:信息倾倒的典型特征

#### 特征1:大段解释性文字
```
❌ 费伦大陆位于世界的北境,气候寒冷。
大陆上有七个王国,星辰法师学院隶属于北境议会管辖。
议会由十二位大法师组成,是北境最高权力机构……
```

#### 特征2:角色对角色解释常识
```
❌ "你知道吗,魔力是宇宙的基本能量,
每个人天生都有魔力通道,但只有少数人能感知到……"
"我知道,这是常识。"
"对,那你还记得魔力分级制度吗?"
```

#### 特征3:作者视角的旁白
```
❌ 在这个世界,魔法是一切的基础。
没有人知道魔法从何而来,也没有人知道魔法的极限在哪里。
法师们用魔法创造奇迹,也用魔法毁灭一切。
```

### ✅ 正确方式的核心要点

1. **Show, Don't Tell** - 展示而非讲述(再次强调)
2. **细节>解释** - 一个具体细节胜过一段解释
3. **行动>陈述** - 让角色"做"而不是"说"
4. **分散>集中** - 信息随剧情自然展开,不集中倾倒
5. **感官>概念** - 让读者"看到""听到""感受到"

- 本书使用费伦大陆，九环法术体系，魔网由米斯特拉维护
- 星辰法师学院：两百余年历史，约三百名在读学徒，按环级分班
- 三年未晋三环 = 取消学籍（这是整个世界的逻辑，不是特例）
- 魔法认知主流：天赋感应论，"与魔法共鸣"，不鼓励用原理分析魔法

## 八、语言风格

### ❌ AI写作痕迹(必须删除)

**标志性AI词汇**:
- 此外、然而、不仅...而且...
- 强调、值得注意的是、需要指出
- 与此同时、总而言之

**四字成语使用原则（问题5确认）**:
- ✅ 可以适度使用(如"头痛欲裂")
- ❌ 不能堆砌/连续使用
- ❌ 避免"头痛欲裂""目光炯炯""锐利如鹰"连续出现

**"的"字密集问题（问题6确认）**:
- 不按数量死卡
- **判断标准**: 读起来是否通顺/别扭
- ❌ "他的眼神的锐利的程度" (明显别扭)
- ✅ "他的眼神很锐利" (通顺)

**句式长度与节奏（问题7确认）**:
- 不按字数死卡
- **核心原则**: 避免用短语/词语简单拼凑
- **追求**: 动作自然连贯、表达简洁准确

**错误示例(机械堆砌)**:
```
❌ 他站起来。走向门口。打开门。走出去。
```

**错误示例(过度修饰)**:
```
❌ 他站起身,径直走向门口。门在他手下打开,他头也不回地走了出去。
```
- 问题:"门在他手下打开"不自然、别扭

**正确示例(自然流畅)**:
```
✅ 他站起身来,推开门,头也不回的走了出去。
```
- 动作连贯
- 表达简洁
- 有细节("头也不回")体现情绪

**核心教训**:
1. **简洁>复杂** - "推开门"比"门在他手下打开"更好
2. **自然>刻意** - 不要为了"显得有文采"而过度修饰
3. **节奏服从内容** - 长短句变化是为了表达需要,不是机械执行

**对话中的数字（问题8确认）**:
- **口语化和阿拉伯数字混用**
- "两天" "三天" (口语化)
- "8点" "25号" (阿拉伯数字)
- **原则**: 怎么自然怎么来

### ✅ 推荐风格总结

- **简洁有力**: 短句为主,长句点缀
- **具体准确**: 用具体词替代模糊词
- **口语自然**: 对话符合人物身份
- **节奏变化**: 紧张用短句,舒缓用长句

### 删除"说明书感"

**问题表现**：
- 元叙事表述："狗血的穿越剧情""套路""网文"等跳出角色的词
- 过度解释推理：把读者能自己理解的推理过程写出来
- 生硬断言："这里肯定不是病房"（单独成段）

**修改方法**：
1. **删除元叙事**
   - ❌ "看来是狗血的穿越剧情发生在我身上了啊"
   - ✅ "高塔？月考？三环法术？爆炸？穿越这种事也能被我碰上？"
   - 原则：用角色视角的关键词连击，不要用作者视角的评价

2. **删减冗余推理**
   - ❌ "明天上午八点？林墨看了一眼窗外的光线，现在应该是清晨六点多，还有不到两个小时……不对，托马斯说的是明天上午。"
   - ✅ 直接删除（读者能自己算）
   - 原则：读者能理解的，不需要主角演示一遍

3. **用疑问替代断言**
   - ❌ "这里肯定不是病房。"（单独成段）
   - ✅ "这难道是什么高级特殊病房？"（接在环境描写后）
   - 原则：疑问比断言更自然，更符合刚醒来的状态

### 关键词连击建立认知

**问题表现**：
- 长篇解释新世界："爆炸对上爆炸？看来是狗血的穿越剧情……"

**修改方法**：
- ❌ "爆炸对上爆炸？看来是狗血的穿越剧情发生在我身上了啊。"
- ✅ "高塔？月考？三环法术？爆炸？穿越这种事也能被我碰上？"
- 原则：用关键词连续疑问快速建立认知框架，比长篇解释更有冲击力

## 九、视角与代入感

### 🔴 核心原则：始终待在主角的意识里

**错误示范（旁白视角）：**
- "林墨感到头痛"
- "他看到石灰岩天花板"
- "意识回来的时候……"
- "第一段浮上来的记忆是……"
- "再往后……"

**正确示范（主角视角）：**
- "头痛欲裂！"（感觉本身就是第一句，不需要"他感到"）
- "石灰岩天花板，裂纹从中间延伸到角落"（天花板直接出现，不需要"他看到"这个中介）
- 脑中闪过失去意识前的画面：CERN实验室……（用"脑中闪过"做缓冲，然后切到记忆内容）
- 删除所有"第一段""先一步""再往后"这种时间导航词，让内容本身推进节奏

### 如何让读者代入主角身份

**1. 用主角的名字做主语**
- ❌ "意识回来的时候……"（谁的意识？抽象的）
- ✅ "林墨知道那是意识还未完全清醒带来的耳鸣"（立刻知道是谁在经历）

**2. 地球记忆和当下感知之间要有过渡**
- ❌ 直接从"头痛"跳到"CERN实验室"（读者会懵：这是回忆还是现在？）
- ✅ "脑中闪过失去意识前的画面：CERN实验室……"（缓冲明确这是回忆，不是旁白）

**3. 加入主角的内心判断**
- ❌ 只描述环境，没有主角的思考
- ✅ "好像是实验发生事故了，那我现在是什么情况。"（思维在动，读者跟着一起推理）

**4. 用动作带场景切换，不用时间标记词**
- ❌ "到这时候，他才注意到……"（旁白在做时间标记）
- ✅ "林墨撑起身来，环顾四周。"（动作本身告诉读者"现在要观察环境了"）

**5. 判断和结论要带主角的语气**
- ❌ "这不是医院。"（客观陈述）
- ✅ "这里肯定不是病房。"（用"肯定"带出林墨的语气，是他的判断）

**6. 记忆触发要有生理反应做缓冲**
- ❌ "右手手背灼了一下，记忆涌进来……"（太生硬，像开关）
- ✅ "右手手背突然灼了一下……紧接着，头部又传来阵阵肿胀感，伴随耳鸣和刺痛，一段记忆片段慢慢浮现"（身体先反应，记忆才来）

**7. 记忆内容要整合，不要碎片堆砌**
- ❌ "金发的脸、走廊、导师、布告栏……"（读者读完还要自己拼）
- ✅ 整合成一段连贯叙述，有因果逻辑："一个月前，原主在公共实验室……"（读者直接理解）

**8. 环境描写要跟着视线走，有层次**
- ❌ 天花板、墙、窗、地板机械扫描式列举
- ✅ 天花板→墙→窗→地板→角落杂物，跟着人的视线自然移动，最后给出判断

### 检查清单

写完一段后问自己：
- [ ] 这段话是"作者在描述林墨"还是"读者在林墨的身体里经历"？
- [ ] 有没有"他看到""他感到""他意识到"这种中介词？（应删除）
- [ ] 有没有"第一段""再往后""到这时候"这种旁白导航词？（应删除）
- [ ] 记忆和现实的切换是否有缓冲？（不能直接跳）
- [ ] 判断和结论是否带着主角的语气和性格？
- [ ] 环境描写是机械扫描还是跟着视线自然移动？
