# 人物设定卡收集文件夹

## 用途
存放人物设定的原始资料、草稿、参考图片等。

## 文件格式
- Markdown 文件（.md）
- 图片文件（.jpg, .png）
- 建议按人物名称命名（如：林墨.md、艾莉娅.md）

## 更新流程
1. 将人物设定保存到此文件夹
2. 运行"更新人物卡"指令
3. AI 会扫描新文件并更新 references/characters.md

## 示例文件结构
```
docs/characters/
├── README.md
├── 林墨.md
├── 艾莉娅.md
├── 托马斯.md
├── 多恩.md
└── images/
    ├── 林墨-参考图.jpg
    └── 艾莉娅-立绘.png
```
