# 读者反馈收集文件夹

## 用途
存放读者反馈的原始记录文件。

## 文件格式
- Markdown 文件（.md）
- 建议按日期或主题命名（如：2026-03-23-反馈.md）

## 更新流程
1. 将读者反馈保存到此文件夹
2. 运行"更新读者反馈"指令
3. AI 会扫描新文件并更新 references/reader-feedback.md

## 示例文件结构
```
docs/reader-feedback/
├── README.md
├── 2026-03-23-第一章反馈.md
├── 2026-03-24-整体反馈.md
└── 语言风格建议.md
```

## 反馈记录模板
```markdown
# [主题/日期]

## 反馈来源
- 来源：[评论区/私信/讨论组]
- 日期：YYYY-MM-DD

## 反馈内容

### 正面反馈
- [具体内容]

### 负面反馈
- [具体内容]

### 建议
- [具体内容]

### 疑问
- [具体内容]

## 关联章节
- 第X章：[相关内容]
```
