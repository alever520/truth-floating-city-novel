## ▌人物卡 #01 — 「铁拳」鲁格

```
种族：半兽人
职业：战士（战斗大师流派）
阵营：守序中立
出身地：艾伯伦大陆，终末战争后的流亡营地
```

**外貌**
身高两米有余，皮肤呈深橄榄绿，下颌突出的獠牙只剩左侧一根（右侧在一场战斗中被打断，他把断牙穿成项链挂在脖子上）。右臂从肩膀到手腕布满烧伤疤痕，战斗时习惯用那只手握武器——"伤过的手更知道怎么打人"。平时穿旧式军用板甲，甲胄上有一处补丁是用不同颜色的金属片修的，他说是"从老战友的铠甲上剪下来的"。

**性格特质**
话极少，但听得极认真。和他说话时你会发现他的眼睛一直盯着你，像是在把你说的每一个字都记下来。对契约和承诺有近乎偏执的执著，答应过的事情不管代价多大都会做到；但他也从不轻易承诺——他说"一句'我会'值一条命"。对弱者有一种粗糙的保护欲，不会说什么温柔的话，只是会默默站到他们前面。

**行为模式**
- 进入新地点时第一件事：找到所有出口，背对墙壁站立
- 吃饭前一定先观察周围的人，确认没有威胁才动筷子
- 受伤时从不表现出来，一个人偷偷处理伤口
- 遇到被欺负的孩子会停下脚步，不说话，只是站在那里直到施暴者离开

**背景故事钩子**
他在终末战争中为雇主战斗了十二年，战争结束后对方用"你只是雇佣兵"为由拒绝兑现承诺的安置费。他没有复仇，只是离开了，因为"打不过的时候要忍，总有打得过的那天"。他现在在找那个雇主，但自己也不确定找到了之后要做什么。

**与队伍的化学反应**
队里的吟游诗人总想给他编故事，他每次都拒绝，但有一次他偷偷在旅馆外面听完了整首关于自己的歌，没有说话就走掉了。

## ▌人物卡 #02 — 薇拉·莫斯泰恩

```
种族：人类（费伦北方血统）
职业：牧师（生命领域）
阵营：守序善良
出身地：费伦大陆，北方小城塞夫罗斯的神庙
```

**外貌**
三十出头，头发是深棕色，常年扎成一个干净利落的低马尾。肤色偏白，脸上有几道因长年在北方风雪中行走留下的细纹，看起来比实际年龄稍老。手指修长，指节处有厚茧——她说是因为"铸锻祈祷书比医治伤口花的时间还多"。神庙制式的白袍总是干净的，即使刚从战场下来也一样，她对此的解释是"这是神的仪表，不是我的"。

**性格特质**
温和但有底线。她会耐心倾听任何人的诉说，但如果有人伤害了无辜者，她的语气会变得像冬天的石头——冷、硬、没有余地。对死亡有一种平静的接受，但对"没有必要的死亡"有强烈的愤怒。她不会把信仰挂在嘴边，但遇到大事时会安静地闭眼祈祷几秒钟再开口说话。

**行为模式**
- 每天清晨五点起床，独自完成一段沉默的祈祷仪式
- 在队伍中会主动记住每个人的伤势和身体状况
- 救治敌人和救治同伴时表情是一样的
- 睡前会在日记本上写当天"应当感谢的三件事"

**背景故事钩子**
她曾经救活了一个刺客——那个人后来杀死了她的师父。她花了整整一年决定这件事是否是自己的错，最终的结论是"不是，但我需要更聪明地行善"。她在找那个刺客，不是为了复仇，而是想亲口问他：你为什么要这么做？

**与队伍的化学反应**
是整个队伍实际上的情绪稳定中心，但她自己从不意识到这一点。每次队伍气氛剑拔弩张时，她只是平静地说"先吃东西"，然后不知为何大家就真的冷静下来了。

## ▌人物卡 #03 — 芬德里克·托奇

```
种族：侏儒（岩石侏儒）
职业：法师（幻术专精）
阵营：混沌中立
出身地：侏儒聚居的地下工坊城市，铜齿山脉深处
```

**外貌**
身高只有九十厘米，但坚持穿一顶高帽来让自己看起来"更有存在感"。皮肤是淡灰色，双眼是罕见的琥珀黄，透出一种与年龄不符的古怪光芒。身上总是挂满各种小玩意儿：水晶球、小镜子、装着奇怪液体的细颈瓶、各种颜色的粉末包——每一个他都能说出一段故事，大部分是真的，小部分是即兴编的。法袍是自己改的，领口绣了一个小章鱼图案，原因他不解释。

**性格特质**
极度好奇，边界感几乎为零。看到任何不熟悉的东西都会立刻凑上去研究，完全不考虑危险性。讲话很快，思维跳跃，对话时经常在中途想起别的事然后跑题。但在施展魔法时会突然变得异常专注，整个人像是换了一个模式——安静、精准、眼神里有一种让人后背发凉的冷静。

**行为模式**
- 睡觉前一定要在脑子里过一遍今天学到的新东西
- 和人说话时眼神总是在飘——不是因为不专心，而是因为同时在观察三件其他的事
- 会把幻觉用在日常生活里：让旅馆显得更豪华，让自己的声音听起来更低沉，让对手误判自己的位置
- 偶尔会在深夜一个人坐着，盯着空气出神，问他在看什么，他说"在看光的折射"

**背景故事钩子**
他有一个老师，一个人类法师，在他还没学完所有知识之前就老死了。他花了二十年在研究"时间魔法"，想要找到一种方法"借"一些时间回去。目前进展是：他知道这不可能，但还是在研究。

**与队伍的化学反应**
是队里最容易制造意外的人——每次他说"我有个主意"，队里其他人都会下意识退后半步。但那些主意平均有七成是真的有效的，所以大家也没法拒绝。

## ▌人物卡 #04 — 阿莎·达尔文

```
种族：提夫林（地狱血统）
职业：邪术师（大魔王契约）
阵营：混沌善良
出身地：费伦大陆，沃特迪普贫民区
```

**外貌**
提夫林特有的小羊角从额头两侧弯曲向上，颜色是深酒红，与她暗紫色的皮肤形成对比。尾巴细长，她平时把它卷在腰间当皮带用。眼睛是没有白眼珠的纯金色——她说这让陌生人害怕，但"这又不是我的错"。穿着刻意普通，破旧的皮夹克，帽子压低，像是在努力让自己不引人注意，但走进一个房间时所有人还是会先看她。

**性格特质**
表面玩世不恭，用幽默和挖苦建立距离感；内里对不公正有一种几乎本能的愤怒。她做任何好事之后都会立刻用一句酸话掩盖，好像不想被人知道自己其实在乎。对契约极为敏感，因为她本人就是一个"契约的产物"——她的祖先和魔鬼签了约，这段血统延续到了她身上。

**行为模式**
- 被人感谢时会立刻转移话题或说一句反讽
- 私下会偷偷接济街上的流浪儿，被人看见就否认
- 在使用契约力量时会短暂出现一种不属于自己的声音，她称之为"房客在说话"
- 对任何形式的控制和束缚有强烈的生理性反应——心跳加速、瞳孔收缩

**背景故事钩子**
她的契约大魔王给她力量，作为交换，魔王偶尔会"借用"她的眼睛观察世界。她不知道魔王在找什么，也不确定自己是否愿意知道。她正在找一种解除契约的方法，但魔王给的力量有时候实在太好用了。

**与队伍的化学反应**
和队里的圣骑士关系最复杂——一个代表神圣秩序，一个带着地狱血统——但两人其实都讨厌不公正，所以打起架来配合得出奇地好。

## ▌人物卡 #05 — 博尔松·铁炉

```
种族：矮人（山地矮人）
职业：战士（防御专精）+ 铁匠
阵营：守序善良
出身地：费伦大陆，铁炉山脉的矮人王国
```

**外貌**
一米四的身高，但肩宽几乎相当于普通人类。红棕色的大胡子编成三股，夹着金属环，其中一个环是他儿子出生时妻子送的。皮肤被锻炉熏成棕褐色，手背和小臂上有几十条或深或浅的金属灼伤痕。他的盾牌自己打的，上面雕着一家人的名字——妻子的、儿子的，还有一个已经用凿子抹掉的名字，他不解释是谁。

**性格特质**
是那种会在最危险的时候背对敌人护住队友的人。极重家庭和血脉，把"家族荣誉"视为最重要的东西——但他自己与家族的关系已经破裂，这个矛盾一直没有解决。不善言辞，但非常善于在沉默中表达立场：他站在谁旁边，就是在说他支持谁。爱喝酒，喝酒之后反而更安静。

**行为模式**
- 队伍扎营时第一件事是检查地面是否适合防守
- 会在闲暇时不由自主地用随手捡到的石头和金属碎片"练手"
- 从不让别人帮他修盔甲，"只有铸造它的人才知道它的弱点"
- 每当路过铁匠铺，都会停下来看一会儿，表情像在回家

**背景故事钩子**
他离开王国是因为当年拒绝了长老的命令——那个命令会让无辜的平民村庄为矿脉让路。他被驱逐，妻儿留在王国。他每年给儿子寄东西，但从来没有收到回音。他不知道是因为信件被拦截，还是因为儿子不想联系他。

**与队伍的化学反应**
是队里最有可能在凌晨三点悄悄去检查营地安全的人，从不声张。队员们慢慢都知道了，但也从不当面提，好像是大家都默契维护的一个小秘密。

## ▌人物卡 #06 — 奈尔·瑟罗斯

```
种族：精灵（高等精灵）
职业：游侠（猎手流派）
阵营：中立善良
出身地：费伦大陆，翡翠森林深处的精灵隐居地
```

**外貌**
精灵特有的轻盈气质，但身上有一种不符合精灵文化审美的粗粝感——皮甲上有爪子抓过的裂痕，靴子底磨得很薄，弓弦换了不止五次，颜色深浅不一的绑扎痕迹像年轮一样记录着岁月。发色是淡银白，剪得比大多数精灵短，说是"长发钩树枝"。左耳上的精灵族标志耳环只剩半个，另一半"在某个猎物嘴里"。

**性格特质**
和自然的亲近是本能的，和人类的亲近是后天学来的，学得七七八八但总有一点点不对劲。比如他理解"玩笑"但时常在错误的时机笑出声；理解"礼貌"但会直接指出对方衣服上的血迹。几百年的寿命让他对时间的感知和普通人类不同——他觉得几年不见"昨天一样"，这无意中伤害了不少关系。

**行为模式**
- 在野外的警觉程度远超在城市里，方向感极佳但在城市里总是迷路
- 会和动物说话（字面意义），回答的内容他自己翻译，准确度存疑
- 对不认识的植物有强迫症式的识别欲望，必须蹲下来观察
- 队伍宿营时他从不睡帐篷，睡在稍远处的树上，说是"听得更清楚"

**背景故事钩子**
他的猎物通缉名单上有一个名字已经跟了三百年。那是一头远古巨型猛兽，毁掉了他所在精灵村落的一半。他追了三百年没追到，现在他也不确定：是猛兽太聪明，还是他其实不想结束这段追逐？

**与队伍的化学反应**
是最难被套话套出信息的人，因为他只说事实，不说感受。但如果队里有人受伤了，他会第二天悄悄放一束有药用价值的草药在那个人的背包里，然后永远否认是自己放的。

## ▌人物卡 #07 — 塔伦·布莱克伍德

```
种族：半精灵
职业：吟游诗人（剑刃学院）
阵营：混沌善良
出身地：费伦大陆，博德之门港口区
```

**外貌**
半精灵的耳朵微微尖，但角度比纯血精灵小，常被人忽略。外形介于人类和精灵之间，长相算是两种审美都沾一点的那种——精灵觉得他太粗犷，人类觉得他太精致。习惯穿色彩鲜艳的衣服，原因是"黑暗里更难被注意到，有趣的地方更容易被邀请"。随身携带的鲁特琴漆面已经磨掉了好几层，把手处缠着一条破旧的蓝色布条——他说是初恋送的，但每次说这个故事细节都不一样。

**性格特质**
天生的故事收集者和故事创造者，对人的秘密有一种职业性的好奇心。但他会判断哪些秘密该说、哪些该藏，这个判断比大多数人想象的要靠谱。嘴里的话有七成是真的，两成是艺术加工，一成是彻头彻尾的即兴创作——他自己分得很清楚，听他说话的人分不清楚。表面上轻浮，但在真正危险的时候会站出来，让人刮目相看。

**行为模式**
- 走进任何酒馆的第一件事是找到情报最多的那个角落坐下
- 和任何人说话时都会找到三个可以称赞的点，不管这个人是什么人
- 会在战斗中唱歌——不是因为不紧张，而是因为这样大家都会跟着节奏行动
- 晚上一个人时偶尔会把琴放下，安静地坐着，表情和白天完全不同

**背景故事钩子**
他曾经在一个贵族沙龙里说了一首批评执政官的讽刺诗，导致三个为他喝彩的普通人被牵连入狱。他花了一年时间把那三个人都救出来，但其中一个没能撑到被救的那一天。他的新诗从来不再批评任何具体的个人——批评的是"制度"。

**与队伍的化学反应**
是第一个发现队员间潜在矛盾的人，也是第一个用一个不经意的笑话化解它的人。他从不直接介入别人的冲突，只是让冲突的两方都觉得自己是对的，然后让时间做剩下的事。

## ▌人物卡 #08 — 格雷芙·怒骨

```
种族：半兽人
职业：野蛮人（狂怒之路）
阵营：混沌中立
出身地：北方冻原，兽人游牧部落
```

**外貌**
身高两米一，肌肉像山峦一样叠起来。皮肤是深绿灰色，脸上有蓝色纹身，那是部落成年仪式留下的，纹路代表他击杀的第一头熊。头发已经在十年前一场战斗中全部烧掉了，从此剃光。右手缺了一根小指，他说是自己切掉的，理由是"它在战斗里妨碍我握斧子"。

**性格特质**
不是"蠢"，而是"直接"——他看穿了社交语言背后的虚伪，于是直接不用。只说必要的话，不掩饰任何情绪，高兴就笑、愤怒就怒，从不用表情撒谎。对强者有发自内心的尊重，对弱者有来自文化底层的蔑视——但他在经历了若干次冒险后，这个"弱"的定义在悄悄改变。

**行为模式**
- 每天早上雷打不动地进行一套源自部落的晨练仪式
- 进城后会把所有绑带和皮甲松开，说是"城里的空气太窄"
- 对陌生人只用"你"称呼，对信任的人会用对方的名字，这是一种荣誉
- 受到重伤时不会叫，但会发出一种低沉的、像野兽一样的喉鸣

**背景故事钩子**
他的部落在一场被一个南方商队引来的怪物袭击中几乎全灭。他是少数存活者之一，因为当时他正在接受一个人独处的成年考验，不在营地里。他带着幸存者找到了那个商队，问清楚了始末，然后离开了——没有杀他们。他不解释为什么没有杀，但他开始思考"仇恨的代价值不值"这个问题。

**与队伍的化学反应**
是队里力气最大、存在感最强、被误解最多的人。但侏儒法师第一个发现他在晚上给武器上油时会轻轻哼一首部落歌谣，此后就成了两个人的秘密。

## ▌人物卡 #09 — 塞拉·卢恩

```
种族：卓尔（黑暗精灵）
职业：法师（塑能专精）+ 逃亡者
阵营：混沌善良（挣扎中）
出身地：幽暗地域，魔索布莱城
```

**外貌**
卓尔特有的黑色皮肤，但因为长年生活在地表，肤色比纯血卓尔略浅了一点，这让她在两个世界都"不够纯正"。白色的长发平时包在黑色布巾里，理由是"太显眼"。眼睛是暗红色，夜里会隐约发光——这让地表的人类总是下意识退一步。身上没有任何代表魔索布莱城的标志，因为她把原来的袍子烧掉了，烧得很彻底。

**性格特质**
极度不信任是她的默认状态，相信某个人对她来说是一件代价极大的事。在城邦里长大意味着每一次相信都可能是陷阱，每一次示弱都可能致命。她在地表花了五年才勉强学会不把每一句关心都解读为威胁。独立性极强，不喜欢欠人情。

**行为模式**
- 不会主动帮助任何人，除非对方问了，并且她判断这个忙值得帮
- 处理危机时比绝大多数人冷静，因为她的日常就是危机
- 对"背叛"的定义比别人严苛——有时候别人觉得只是误会，她已经拔刀了
- 有一本加密的手账，里面记录着她认识的每个人以及她对他们的"信任评分"

**背景故事钩子**
她逃出魔索布莱城的理由是拒绝参与一次对无辜者的祭祀仪式。她的母亲下令追杀她，她的妹妹负责执行追杀命令。她一直不知道妹妹是真的信仰那套价值观，还是只是没有选择。

**与队伍的化学反应**
和牧师的关系是全队最微妙的——牧师代表的神曾经是她被告知要憎恨的存在，但此刻那个神的信徒救了她的命不止一次。她正在用一种非常缓慢的速度，重新评估这件事。

## ▌人物卡 #10 — 朵朵·瑞弗韦斯

## ▌人物卡 #11 — 艾利安·达恩

```
出身地：费伦大陆，银月森林边缘
```

月精灵的皮肤透出淡淡的银蓝光泽，在满月的夜晚会更明显。眼睛是浅金色，瞳孔竖着，像猫。头发长且白，常年散着，里面缠绕着几根细树枝和一根羽毛——羽毛来自一只已经死去的白鹰，是他的第一个"动物伙伴"。穿着用植物纤维编织的长袍，没有任何金属部件，理由是"自然不喜欢金属的声音"。

思考时间尺度和正常人不同——他关心的问题是"这片森林一百年后会怎样"，而不是"这场战斗今晚会赢吗"。这让他在短期决策上显得迟钝，在长期判断上却往往惊人地准确。他不评判善恶，他评判"是否符合自然的秩序"，这让他有时候看起来冷酷，有时候看起来包容。

- 变身成动物时性格有细微变化，变成鸟时话更少，变成狼时警觉性更高
- 从不伤害植物，绕路也要绕
- 对"消耗自然资源"的人类行为有低鸣式的持续不满，但他从不直说，只是眼神会变

**背景故事钩子**
银月森林曾经有一块区域被人类矿工砍伐，他试图用德鲁伊的方式阻止，失败了。愤怒驱使他做了一件事——他把那片区域的土地诅咒了，让它二十年内寸草不生。现在他一直在等那片土地再次发芽，像是在赎罪。

**与队伍的化学反应**
和侏儒法师的关系最有趣——一个研究人工魔法，一个信仰自然魔法——两个人对"魔法的本质是什么"这个问题有完全相反的看法，但吵架吵了那么多次，彼此其实都在悄悄修正自己的观点。

## ▌人物卡 #13 — 韦尔·托斯坦


把死亡当作学术课题研究，这让大多数人不舒服，但他对此无法理解为什么——"死亡是世界上最普遍的现象，不研究才奇怪"。道德上实际上没有想象中邪恶，他只是把"研究对象的意愿"排在"获取知识"之后，这是他唯一真正的问题。

- 会不经意间说出一些在他看来是学术事实但对普通人来说很可怕的话

他的故乡小镇曾经被一支不死军团摧毁，他是少数幸存者之一。他没有去复仇，他去研究了——研究是谁创造了那支军团，以及怎么永久摧毁死灵魔法。他的研究有了一些进展，但他越来越发现"摧毁死灵魔法"和"彻底理解死灵魔法"之间的边界正在模糊。

**与队伍的化学反应**
和牧师（生命领域）的关系是哲学对立——一个认为生命是神圣的，一个认为生死是可以被研究和干预的。他们讨论的时候队里其他人都会停下来听，因为两个人都有道理，但两种道理加起来不能共存。

## ▌人物卡 #15 — 艾西娅·温纳斯

高原游牧民的世界观：移动才有生路，停下来就是死亡；强者保护弱者是义务，不是善心。她对"家园"的概念是移动的——家是你的马和你的同伴，不是一块地。这让她在队伍中有时候显得"冷静得可怕"，因为她真的不怕失去一个地方。

- 睡觉时永远侧卧，背不贴地，据说是游牧民的习惯，便于迅速起身

**与队伍的化学反应**
和矮人战士是默契最高的战斗搭档，原因很简单：两个人都喜欢少说话多行动，彼此都信任对方的判断，在战斗中互相补位像呼吸一样自然。在营地里两个人也经常一起坐着，什么都不说，这被队里其他人称为"最高规格的交流"。

## ▌人物卡 #16 — 奥里恩·格雷斯托姆

**与队伍的化学反应**
和吟游诗人的关系最微妙——吟游诗人见过各种各样的人，不把贵族头衔当一回事，这对奥里恩来说既是冒犯又是一种奇怪的解放。他们经常辩论，但总是辩着辩着换成聊别的事了。

## ▌人物卡 #17 — 苔絮

```
种族：精灵（木精灵，野生型）
职业：游侠（猛兽守护者）

```

木精灵的皮肤是暗棕色，带有天然的迷彩效果，在树丛中几乎看不出来。头发是深绿色，里面真的长着几片小叶子——她说不是装饰，是自己长出来的，这令其他人半信半疑。身材纤细但极具爆发力，静止时像一棵树，移动时像风。眼睛是琥珀色，有很强的环境感知能力，但不善于解读人类的表情。

**行为模式**
- 召唤兽（一只独角瘦狼）是她最重要的存在，比任何队友都优先
- 不理解为什么要在室内睡觉，但为了队伍会勉强配合，但要开着窗
- 接受帮助的方式是下次在对方不注意时带回一只受伤的小动物让他们帮忙照顾
- 用气味判断一个人的情绪，"你闻起来很怕"比"你看起来很怕"更常从她嘴里说出来

**背景故事钩子**
她曾经目睹猎人捕杀了一种极为罕见的白色鹿，那头鹿在她的文化里是神明的信使。她杀掉了那个猎人。她一直没有决定那件事是对是错，而且随着时间推移这个问题变得更难回答，不是更容易。

**与队伍的化学反应**
和半身人盗贼是出乎意料的好友——两个人都相信直觉多过计划，都有一种孩子式的对世界的好奇心，一个用捡东西表达，一个用观察生物表达。在队伍里她们俩是唯一真正静得下来的人。

## ▌人物卡 #18 — 达文·莫雷尔

```
种族：人类（东方行商家族出身）
职业：吟游诗人（秘识学院，间谍型）
阵营：中立（以"信息中立"为旗号，实际上暗中偏善）
出身地：费伦大陆，海港城市沿途商路
```

**外貌**
三十八岁，长相普通——是那种会在人群中消失的普通，让见过他的人一小时后就想不起来他长什么样。这不是魔法，是技术：他在不同场合的服装、语气、走路姿势都不一样，像是一个装着十个人的容器。只有眼睛是一直不变的：深棕色，看一个人时像是在读一本书。

**性格特质**
信息是他的货币、武器和安全感的来源。他永远比别人知道更多，但很少全部说出来——说出来就贬值了。他自己对自己的道德立场有清晰的认知：他不在乎善恶，在乎的是"均衡"。但他有一个他不承认的倾向：当选择是让强权稳固还是让弱者有机会时，他总是会把那个"一点点"的信息差递给后者。

**行为模式**
- 在任何社交场合同时参与三段不同的对话
- 第一次见面必定会问三个看似闲聊但精心设计的问题
- 有一个习惯：结束一段重要对话时会在心里默数五秒，观察对方的第一个自然反应
- 记账本里夹着各地地图，每个城市标注的不是景点而是"信息密度最高的位置"

**背景故事钩子**
他曾经在一次情报交易中，因为没有及时透露一条关键信息，导致一个小国无力防御，被一个大国吞并。三万人因此流离失所，其中有些人他认识。他把那条信息卖给了最高出价者，那是他职业规则里完全合理的操作。从那以后他开始给自己立了一条只有自己知道的补充规则。他没有告诉任何人这条规则是什么。

**与队伍的化学反应**
和死灵法师的关系是全队最奇怪的友谊——两个人都以"客观"自居，都把情感压在信息处理能力之下，都知道自己在撒谎。他们之间的沉默对话有时候比其他人的长篇大论传递更多。

## ▌人物卡 #19 — 嘉雅·法林

```
种族：提夫林（古魔帝国血统）
职业：牧师（死神领域，非邪恶解读）
阵营：中立善良
出身地：费伦大陆，边境旧战场附近的村庄
```

**外貌**
暗紫色皮肤，但比阿莎更深，几乎接近黑色。羊角更大更弯，她用金线绕在角上，不是装饰，是一种古代提夫林守护仪式留下的习俗。眼睛是白色的，瞳孔是深灰的，看起来像盲人，但视力完全正常。脸上永远带着一种平静，这种平静不是压抑情感，而是"在情感之下有一种更稳定的东西"。

**性格特质**
她信奉的不是"让人死"的死神，而是"让死亡有意义"的死神——这个区别对她来说比天和地还大，对大部分人来说难以理解。她是那种会陪在垂死之人身边、让他们在平静中离去的人，这件事她做得比任何治疗都认真。对生命极度珍视，恰恰因此对"无意义的死亡"有最强烈的愤怒。

**行为模式**
- 在战斗结束后，无论多疲惫都会先检查敌我双方的全部伤亡
- 会为死去的敌人做一个简短的安息仪式，哪怕只是在心里
- 她的神术倾向于"终结"——终结痛苦、终结诅咒、终结不死状态
- 会问垂死之人"有什么话想说"，然后记下来，找机会传达给对方想传达的人

**背景故事钩子**
她的村庄曾经被一个死灵法师用不死魔法控制，村民们变成了行尸走肉。她的父母也在其中。她无法救他们，但她把每一个被不死力量控制的村民都释放了，包括父母。这个行为在她信奉的神明看来是一种祭献，也是一种宣誓。

**与队伍的化学反应**
和死灵法师的存在构成完全不同的对立——不是敌对，而是价值观的镜像。他研究死亡，她尊重死亡。他想改变死亡，她想让死亡有尊严。他们彼此能理解对方的逻辑，但无法接受对方的结论，这让两人的对话总是以沉默收场，但那种沉默是有重量的。

## ▌人物卡 #20 — 伊多·瑟兰

```
种族：侏儒（森林侏儒）
职业：盗贼（假面大师 + 幻术辅助）
阵营：混沌善良
出身地：费伦大陆，侏儒艺人巡演车队
```

**外貌**
森林侏儒，身高八十厘米，皮肤是棕绿色，在草地上天然隐蔽。眼睛是绿色，非常大，占脸比例有点超标，让他看起来永远像是刚刚见到什么惊喜。随身携带一个巨大（相对他体型而言）的手提箱，里面是他的全部家当：三十七个面具，各种颜色的假发，七种不同的靴子（用来模拟不同步伐），以及各种用于变装的道具。他自己只有在睡觉时才"还原成本人"，其他时候永远在扮演某个角色，连队友有时候忘记他本来是谁。

**性格特质**
从来不撒谎，但永远在扮演角色——这个区别对他来说非常重要。角色说的谎不是他的谎，这个道德逻辑别人不接受，他坚持认为它成立。热爱人类（广义的"人类"，涵盖所有种族），热爱故事，热爱变化。对真实身份有一种复杂的依恋和逃避——他躲在面具后面不是因为自己不好，而是因为"自己"是他还没想清楚的东西。

**行为模式**
- 走进一个新地方的第一件事：确定这里最需要哪种"角色"
- 扮演一个角色扮演得太久时会出现短暂的身份混乱，需要独处一小时"找回自己"
- 他的真名只告诉了两个人，每次问他为什么挑那两个人，他说"感觉对了"
- 会在夜里取下所有面具，坐在镜子前看自己，时间长短不一，看完后通常睡得很沉

**背景故事钩子**
巡演车队在他十六岁时遭遇了一伙强盗。所有成年人都被抢劫，但因为他当时恰好在扮演"另一个人"，强盗没有注意到他。他就那样躲着，看着同伴被抢，没有出声。他现在不确定那个决定是智慧还是懦弱，但他从那以后开始认真学习一切可以学习的战斗和生存技能，"下次不一样"。

**与队伍的化学反应**
是队里唯一一个能让所有其他成员都放松下来的人，原因很简单：他用三十七张脸看过了三十七种世界，没有什么判断是他发出的。和他在一起，人会觉得被看见的同时也被接纳。队里的矮人战士曾经喝多了悄悄说："如果我有孩子，我希望他像你。"伊多那天晚上摘下了面具，安静地坐了很久。
