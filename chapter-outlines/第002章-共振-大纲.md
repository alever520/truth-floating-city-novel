# 第002章《共振》章节大纲

**核心任务**
林墨分析原主的失败数据，发现魔力输出曲线的"周期性波动"，提出"共振假设"。进行三次验证实验：第一次摸索固有频率，第二次失控导致魔力反噬，第三次成功达到三环质感。托马斯目睹全过程，震惊与疑虑加深。

**核心冲突**
- **表层冲突**：在一天内验证假设，确保明天月考能通过
- **深层冲突**：托马斯察觉到林墨的思维方式完全变了（记笔记、分析数据、提假设），内心挣扎加剧
- **戏剧张力**：林墨专注于实验，没有意识到托马斯的情绪变化

**预计字数**：3500-4000字

## 场景规划

### 场景一：分析笔记（宿舍，早上7点30分）

林墨翻出原主的黑皮笔记本，三年失败记录密密麻麻，最后一页只有一行：*强行压缩，冲击三环——明天试。*

笔记里夹着托马斯手抄的控制技巧小册子，书脊是新的——原主从未翻过。

林墨把两本笔记摊开对照，开始用物理学家的方式分析原主的失败数据。

他很快发现一个被忽略的细节：

**关键发现**：
```
原主每次失败时，魔力输出曲线都有"异常波动"
不是随机波动，而是有固定的周期性——大约1.2秒一个周期

林墨突然想到物理课上学过的"共振"：
- 每个系统都有固有频率
- 如果外力频率与之匹配，就能产生共振
- 用很小的力就能达到很大的振幅（推秋千原理）

假设：魔力回路也有"固有频率"
如果能找到这个频率，顺着它推动魔力
就能用更少的魔力达到三环的密度
```

林墨在笔记上记录：*假设：魔力回路固有频率约1.2秒。需验证。*

托马斯坐在一旁，看着林墨记笔记的样子，眼神更加复杂了。

"你以前……从来不记笔记。"托马斯突然开口。

林墨抬头："嗯？"

"你以前不记笔记，不分析数据，不说'假设''验证'这种词。你……"托马斯停顿了一下，"你真的变了。"

林墨沉默几秒，没有否认。

**关键对话**：
> 托马斯："你以前从来不记笔记。"  
> 林墨："……现在记还来得及。"  
> 托马斯（盯着他）："你真的变了。"

**关键细节**：
- 原主从未翻过托马斯的手抄小册子（体现托马斯的付出与原主的冷漠）
- 林墨的系统性分析方法（记录、对比、提假设）
- 托马斯察觉到思维方式的根本改变

**字数**：约800-1000字

---

### 场景二：第一次验证（宿舍，早上8点）

林墨站起来，走到宿舍中央的空地。

托马斯抬头："你要做什么？"

"验证一个假设。"林墨活动了一下手腕，"如果假设成立，我明天就能通过月考。"

**第一次尝试（摸索）**：
```
林墨闭眼，用很轻的力"试探"
让魔力自由流动，不加任何干预
只是感受它的"自然节奏"

他感觉到了——
魔力像波浪，有高峰、有低谷
周期大约是1.2秒
和笔记里记录的周期性波动完全一致

但密度只到二环
他还没找到正确的推动方式
只是确认了"固有频率"的存在
```

林墨睁开眼，在笔记上记录：*验证1：固有频率1.2秒，确认假设成立。但密度只到二环，需找到共振方式。*

托马斯看着林墨写字，欲言又止。

**关键细节**：
- 林墨的实验方法（先摸索，再记录）
- 确认了"固有频率"的存在
- 托马斯的沉默观察

**字数**：约600-800字

---

### 场景三：第二次验证（宿舍，早上8点10分）

**第二次尝试（失控）**：
```
林墨开始在每个"波峰"时轻轻推一把
不是用蛮力，而是顺着那个节奏
像推秋千一样

魔力开始"荡起来"，越来越高
密度逐渐接近三环

林墨心跳加速——快成功了！

但就在那一瞬间——
他的节奏乱了！
推动时机偏差了0.3秒，在波谷时推了一把
打乱了整个节奏

魔力反噬！

手背剧痛，像被针刺火烧
旧伤泛起淡淡红光，温度骤升
```

托马斯惊醒，脸色煞白："你没事吧？！"

林墨咬牙停手，冷汗顺着额头滑落。魔力波动在体内乱窜，他花了十几秒才平复下来。

"没事。"他深呼吸，擦掉额头的汗，"我知道问题在哪里了。"

托马斯犹豫了一下："要不要休息一下？明天还要月考，万一伤得更重……"

"不用。"林墨在笔记上记录：*验证2：失败。推动时机偏差0.3秒导致共振中断。结论：必须精确控制，误差<0.1秒。*

托马斯看着林墨记笔记的样子，眼神更加复杂了。震惊、不解、恐惧，还有一丝……羡慕？

**关键对话**：
> 托马斯："要不要休息一下？明天还要月考……"  
> 林墨（擦汗）："不用。我必须今天搞清楚。时间不多了。"  
> 托马斯（沉默，看着林墨记笔记）："……"

**关键细节**：
- 魔力反噬的代价（手背剧痛、冷汗）
- 林墨即使失败也记录数据（科学家本色）
- 托马斯的情绪变化（震惊→恐惧→羡慕）

**字数**：约800-1000字

### 场景四：第三次验证（宿舍，早上8点25分）

**第三次尝试（成功）**：
```
林墨吸取教训：
必须全程保持节奏，精度要在±0.1秒以内
这需要极高的专注度

他再次闭眼，找到那个1.2秒的周期
这次他更加专注，像指挥家打拍子
在脑中数数："一、二、推。一、二、推。"

每次推动都精确卡在波峰

魔力越荡越高
像秋千被推到了最高点

那一瞬间，他清楚感受到三环的"质感"
——更沉、更满、更稳定
就像从液体变成了固体，密度完全不同

成了！

而且他感觉到：
自己的魔力只用了平时的60%
剩下的40%都是魔力回路自己"共振"出来的

这就是共振的力量
```

林墨睁开眼，手心的魔力缓缓散去。他看着自己的手，第一次觉得"魔法"这个词如此清晰。

**林墨的内心独白**：
> "原来魔法也遵循物理规律。  
> 不是'神秘的共鸣'，而是'可计算的共振'。  
> 只要找到正确的频率和节奏，任何人都能做到。  
> 这不是天赋，是方法。  
> 而且——这个方法可以被传授。"

林墨在笔记上记录最后一行：*验证3：成功。魔力消耗约60%，密度达到三环标准。结论：共振假设成立。明天月考可以通过。*

托马斯站在旁边，眼神复杂——震惊、不解、恐惧，还有一丝羡慕。

"你……刚才做到了？"

"做到了。"林墨看着自己的手，"我找到方法了。"

托马斯张了张嘴，又闭上。手指攥紧了袍子下摆，往后退了半步。

停顿了很久，他才开口："你以前……从来没有这么说过话。"

林墨抬头。

托马斯移开视线："你以前不记笔记，不分析数据，不说'假设''验证''共振'这种词。你……"

他停顿了一下，声音很低："你真的还是林墨吗？"

林墨沉默了几秒。这已经是托马斯第二次问这个问题了。

"我还是我。"林墨看着托马斯的眼睛，"只是……昏迷三天，想明白了一些事情。"

托马斯盯着他，像在确认什么。最后他只是点了点头，没再说话。

但他的手一直攥着袍子下摆。

**关键对话**：
> 托马斯："你……刚才做到了？"  
> 林墨："做到了。我找到方法了。"  
> 托马斯（沉默很久）："你以前……从来没有这么说过话。"  
> 林墨："我还是我。只是……想明白了一些事情。"  
> 托马斯（盯着他看）："……"

**关键细节**：
- 成功的喜悦与代价（魔力消耗60%，但达到三环）
- 林墨意识到"方法可以被传授"（研究线起点）
- 托马斯第二次质疑，更加直接
- 两人之间的张力达到顶点

**字数**：约1200-1500字

### 场景五：准备出发（宿舍，早上8点40分）

窗外光已经很亮。托马斯站起来整理袍子，转过身神色复杂："你今天……不对，你这三天醒来之后，看起来完全不一样了。"

林墨愣了一下："什么意思？"

"你变得……更冷静，更有逻辑。你会记笔记，会分析数据，会用'推秋千'来解释魔法。"托马斯停顿了一下，"你以前不是这样的。"

林墨沉默几秒："人总会变的。"

"会变，但不会变得这么彻底。"托马斯深吸一口气，"但不管你是谁……我陪你去月考。"

手指又在袍子下摆上攥了一下。

**关键对话**：
> 托马斯："你变得更冷静，更有逻辑。你以前不是这样的。"  
> 林墨："人总会变的。"  
> 托马斯："会变，但不会变得这么彻底。"（停顿）"但不管你是谁……我陪你去月考。"

**关键细节**：
- 托马斯的矛盾：怀疑 vs 忠诚
