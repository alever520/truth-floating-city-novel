# 第003章《月考》章节大纲

**核心任务**
林墨前往考场参加三环晋级月考，在法阵中展示"谐振放大"技术。施法过程中，魔力呈现周期性波动，三位教授注意到异常。最终以90分通过（耗魔降低39%），但正是因为表现太好，引起学院的"持续观察"决定。

**核心冲突**
- **表层冲突**：通过三环晋级月考（林墨的目标）
- **深层冲突**：改良后的施法方式过于优秀，引发学院"异常关注"
- **戏剧反讽**：林墨以为"通过考核就安全了"，但正是因为"表现太好"而被盯上

**预计字数**：3500-4000字

## 场景规划

### 场景一：去往考场（路上，早上8点10分）

两人推开门，走上石板路。

石板路，尖顶建筑，晨光金黄。路上遇到学徒，看到林墨都停下脚步，目光复杂。有人甚至下意识后退半步，像在躲避什么不祥的东西。

经过医疗院，二楼窗边站着穿白袍的老者——头发全白，面容枯槁，目光锐利得像鹰。他正盯着林墨，像在观察实验样本，眼神里没有温度，只有纯粹的学术好奇。

林墨和他对视一秒，老者微微点头，转身离开。

托马斯拉了拉林墨袖子："别看了,快走。"

"那是谁？"

"医疗院的霍尔教授。他是学院最资深的治疗师，也是……灵魂学的专家。"托马斯压低声音，"他来看过你，拿着那个水晶球检查了很久。我听到他和卡洛斯导师说……"

托马斯停顿了一下，没有继续。

"说什么？"

"没什么。"托马斯加快脚步，"快到时间了。"

林墨没再追问，但他记住了这个名字——霍尔教授，灵魂学专家。

**关键细节**：
- 其他学徒的异样反应（好奇+警惕+退避）
- 霍尔教授的锐利目光（像观察实验样本）
- 托马斯欲言又止（暗示有重要信息未透露）
- 学院氛围的压抑感

**字数**：约600-800字

### 场景二：考场门外（主楼三层，早上8点25分）

厚重橡木门，门外站着两个高年级执事，都穿着深蓝色长袍，胸前绣着学院徽章——一座高塔被星环环绕。

其中一个看到林墨，愣了一下，对同伴低声说了什么。

托马斯递上林墨的学徒证明，执事检查后推开门。门很重，发出低沉的吱呀声。

托马斯握了握林墨的肩膀："别紧张，你可以的。"

手很用力，像在传递某种力量，又像在做最后的告别。

林墨点头，推门进去。

**考场环境**：
- 房间不大，大概三十平米
- 中央是直径三米的法阵，银色纹路精密复杂，在晨光中微微发光
- 墙边坐着三位教授，都穿绣金边的深红色法袍：
  - **左侧**：中年女性，黑发，面无表情，手持羽毛笔和羊皮纸（记录员）
  - **右侧**：格雷文，花白头发梳得一丝不苟，眼神锐利如鹰
  - **中间**：霍尔教授，手里放着羊皮纸和怀表，目光平静

林墨行学徒礼——右手握拳放在左胸，微微鞠躬。

格雷文微微点头："林墨·阿什顿，三年级学徒，今日接受三环晋级考核。"

他顿了一下，声音低沉："考核科目是三环塑能系法术：**火球术**。"

"你需要在法阵中央凝聚火球，保持稳定三秒，然后精确击中五米外的靶心。我们会根据精准度、稳定性、耗魔量三个维度评分。六十分及格。"

停顿。

"过去三年，通过率约为35%。"

格雷文看了一眼怀表："你有三十秒准备时间。之后进入法阵开始施法。"

"现在——计时开始。"

**关键对话**：
> 格雷文："林墨·阿什顿，三年级学徒，今日接受三环晋级考核。"  
> （停顿）  
> "考核科目是三环塑能系法术：火球术。"  
> "过去三年，通过率约为35%。"

**关键细节**：
- 执事的异样反应（对林墨的特别关注）
- 托马斯"最后的告别"般的握手
- 这是正常的考核流程，没有任何"陷阱"的迹象
- 霍尔教授在场（灵魂学专家）
- 格雷文的威严与专注

**字数**：约600-800字

### 场景三：施法过程（法阵中，8点28分）

**准备阶段（30秒）**：
```
林墨踏入法阵，闭眼
手指轻微颤动，像在计数

他在寻找魔力回路的"固有频率"
刚才在宿舍验证时是1.2秒
但每个法术的频率可能不同
他需要重新确认

感受……感受……

找到了！
1.2秒的周期
和早上验证时一模一样
看来这是他的魔力回路的固有频率
不是法术本身的频率
```

**教授们的观察**：
```
格雷文低声："他在做什么？手指一直在动。"
霍尔教授："好像在……数拍子？"
记录员已经开始记录："准备阶段，候选人闭眼，手指有节奏颤动，周期约1.2秒。行为异常，疑似紧张反应。"
```

三十秒很快过去。

格雷文："时间到。开始施法。"

**施法开始**：
```
林墨睁眼，开始凝聚魔力

但他的动作很奇怪——
不是传统的"持续推动"
而是"推一下，停一下，再推一下"

每次推动的间隔都是1.2秒
像心脏跳动一样有节奏

记录员低声："魔力输出不连续……呈现周期性波动，周期1.2秒。"

格雷文皱眉："这不符合标准施法模式。魔力中断会导致法术散掉。"

但火球没有散
反而越来越大、越来越稳定

林墨的魔力输出量明显低于标准值
——法阵的探测符文显示他的输出功率只有标准值的61%——
但火球的密度已经达到三环标准

霍尔教授眼睛微微睁大："他在利用某种……共振效应？"

格雷文看向霍尔："什么意思？"

"我不确定。"霍尔盯着法阵中央，"但他的施法方式……像是在和魔力回路共振。每次推动都卡在某个特定时机，魔力自己在放大。"

记录员快速记录："疑似共振效应，输出功率61%，密度达标。"
```

**火球成型与发射**：
```
三秒后
火球完美成型

橙红色，直径30厘米，温度稳定在标准范围内
表面没有任何波动，像是一个完美的球体

林墨手一挥
火球精准击中五米外的靶心

爆炸，威力标准，范围可控
烟尘散去，靶心正中央烧出一个完美的圆形焦痕

法阵的魔力探测符文闪烁
数据自动记录在羊皮纸上
```

**关键细节**：
- 林墨的"数拍子"动作（教授们注意到异常）
- 周期性波动的施法方式（不符合标准模式）
- 魔力输出只有61%，但密度达标（矛盾）
- 霍尔教授敏锐地察觉到"共振效应"
- 完美的施法效果（98分精准度，96分稳定性）

**字数**：约1200-1500字

### 场景四：结果公布（考场，8点30分）

记录员看着羊皮纸，眉头微皱。停顿了两秒，抬头看向格雷文。

格雷文："念。"

记录员："精准度：98分。靶心正中，偏差不到1厘米。"

"稳定性：96分。火球密度均匀，表面波动<3%。"

停顿。

"耗魔量……"她又看了一眼羊皮纸，像在确认自己没有看错，"比标准值低39%。"

沉默。

三位教授同时看向林墨。

格雷文眼神一凝，看向霍尔教授。

霍尔教授翻开林墨的历史档案——那是一本厚厚的羊皮册子，记录了三年的施法数据："林墨·阿什顿，入学第一年，施法效率：标准值102%。第二年：标准值108%。第三年上半年：标准值105%。"

他抬头："过去三年的施法效率记录，从未低于标准值，多数时候略高于标准值。"

格雷文盯着林墨："你的施法方式……是谁教你的？"

林墨如实回答："没有人教。昏迷三天醒来后，我突然感觉到魔力有'节奏'，就试着顺着那个节奏推，结果发现省力很多。"

格雷文和霍尔对视一眼。

格雷文："你是说……你自己摸索出这个方法的？"

林墨点头："是的。我翻了笔记，发现我以前失败时，魔力输出曲线有周期性波动。我猜测这是魔力回路的固有频率，所以试着利用这个频率，结果成功了。"

记录员迅速记录林墨的每一个字："候选人自述：昏迷后顿悟，感知到魔力'节奏'，提出'固有频率'假设，自行验证成功。"

格雷文沉默几秒："总分：90分。通过。你可以离开了。"

林墨再次行礼，转身离开。

厚重的橡木门关上。

**关键对话**：
> 记录员："耗魔量……比标准值低39%。"  
> （沉默）  
> 格雷文："你的施法方式……是谁教你的？"  
> 林墨："没有人教。昏迷三天醒来后，我突然感觉到魔力有'节奏'……"  
> 格雷文："你是说……你自己摸索出这个方法的？"  
> 林墨："是的。我猜测这是魔力回路的固有频率，所以试着利用……"

**关键细节**：
- 耗魔量降低39%（接近40%）
- 林墨主动解释了原理，认为这是"顿悟"
- 格雷文没有表现出怀疑，但记录很详细
- 三位教授的眼神交流（评估这件事的意义）
- 林墨不知道自己的解释"太清晰了"

**字数**：约800-1000字

### 场景五：走出考场（门外，8点32分）

林墨走出考场，托马斯在门外等着。看到林墨出来，眼神复杂："怎么样？"

林墨点头："通过了。90分。"

托马斯松了口气："太好了！"

他停顿了一下，声音很低："我还以为……"

"以为什么？"

"没什么。"托马斯移开视线，"我们回去吧。"

两人沉默着走回宿舍。

**林墨的视角（内心独白）**：
> "通过了。  
> 原来这个世界的魔法也遵循物理规律。  
> 只要找到正确的方法，任何人都能进步。  
> 接下来，我要好好研究这个世界的'法则'。  
> 如果魔法可以被理解、被优化、被传授……  
> 那这个世界会发生什么变化？"

**关键细节**：
- 林墨以为"通过考核就安全了"
- 托马斯松了口气，但仍有隐忧
- 林墨开始思考"魔法的本质"和"传授的可能性"
- 镜头切换到考场内（为下一场景做铺垫）

**字数**：约300-400字

### 场景六：考场内的讨论（考场，8点35分）

厚重的橡木门关上，隔绝了外面的声音。

三位教授围坐在桌前。

格雷文翻开林墨的档案，那是一本厚厚的羊皮册子："林墨·阿什顿，入学三年，资质评定：C+，施法天赋：标准偏下。家境：平民，靠助学金维持。"

他翻到最后一页："第35个月，强行施放三环火球术，魔力反噬，昏迷三天。"

霍尔教授："昏迷三天后，突然掌握了从未见过的施法技巧。"

记录员:"他说是'顿悟'。这种案例不是没有……魔力反噬有时会刺激灵魂深处的潜力，导致顿悟。"

格雷文打断："但从未有人能在顿悟后立刻将耗魔量降低近40%。"

他停顿了一下："这不是顿悟，这是……系统性的方法。"

霍尔教授点头："他提到了'固有频率'、'周期性波动'、'共振'……这些词汇不在我们的魔法理论体系里。我从未听说过有人用这种方式理解魔法。"

记录员："他还说'翻了笔记，分析数据，提出假设，验证成功'。这种思维方式……"

格雷文用羽毛笔在档案上写下一行字：

**"第36个月，施法效率异常提升，耗魔量降低39%。疑似掌握某种未知优化方法。建议持续观察。"**

他抬头看向另外两人："接下来的几周，记录他所有的施法行为。我要确认……这是巧合，还是某种可复制的技术。"

霍尔教授："如果是后者……"

格雷文："如果是后者，这将改变我们对魔法的认知。"

停顿。

格雷文合上档案，看向窗外。阳光照在学院的尖塔上，金光灿烂。

"先观察。不要打草惊蛇。"

**关键对话**：
> 格雷文："这不是顿悟，这是……系统性的方法。"  
> 霍尔教授："他提到了'固有频率'、'周期性波动'……这些词汇不在我们的魔法理论体系里。"  
> 格雷文（写下记录）："建议持续观察。"  
> "如果是后者，这将改变我们对魔法的认知。"  
> "先观察。不要打草惊蛇。"

**关键细节**：
- 格雷文判断这是"系统性方法"，不是顿悟
- 林墨的思维方式与传统法师完全不同
- 学院决定"持续观察"（不是立刻行动）
- "改变魔法认知"的可能性
- 格雷文的审慎态度（先观察，不打草惊蛇）

**字数**：约600-800字
